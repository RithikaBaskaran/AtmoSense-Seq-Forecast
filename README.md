# AtmoSense-Seq-Forecast

Multi-Pollutant Air Quality Forecasting Using a Sequence-to-Sequence Transformer Neural Network

## Project Structure
- `src/` — Python source files (one per team member)
- `notebooks/` — Jupyter notebooks for exploration and development
- `results/` — Evaluation metrics and outputs
- `data/` — Download instructions only (dataset stored on Google Drive)

## Team
- Person 1 — Data Engineer (dataset.py)
- Person 2 — Model Architect (model.py)
- Person 3 — Training Engineer (train.py)
- Person 4 — Evaluation & Visualization (evaluate.py)
